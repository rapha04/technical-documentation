Introduction to JavaScript

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language. Inside a host environment (for example, a web browser), JavaScript can be connected to the objects of its environment to provide programmatic control over them.

JavaScript contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements. Core JavaScript can be extended for a variety of purposes by supplementing it with additional objects

JavaScript is an interpreted language, which means that it doesn't need to be compiled.

It is an essential component of modern web applications.

It is used for developing interactive and dynamic websites.

console.log('Hello, World!'); let num = 5; const PI = 3.14159;

    • Variables

    You use variables as symbolic names for values in your application. The names of variables, called identifiers, conform to certain rules.

    A JavaScript identifier must start with a letter, underscore (_), or dollar sign ($); subsequent characters can also be digits (0-9). Because JavaScript is case sensitive, letters include the characters "A" through "Z" (uppercase) and the characters "a" through "z" (lowercase).

    The var keyword is used to declare a variable globally or locally to an entire function.

    The let keyword allows you to declare a variable with block scope.

    The const keyword is used to declare a constant whose value cannot be changed once it has been assigned.

    let name = 'John'; var age = 30; const PI = 3.14159;

    • Functions

    Functions are one of the fundamental building blocks in JavaScript.

    A function is a block of code that can be defined and then called at a later point in the program.

    A function definition (also called a function declaration, or function statement) consists of the function keyword

    Functions can take parameters, which are values that the function uses to perform its tasks.

    function greet(name) { return "Hello, " + name + "!"; } let result = greet('Alice');
  • Loops
  • Arrays
    • Loops

    Loops are used in JavaScript to execute the same block of code multiple times.

    There are different types of loops in JavaScript, such as for loops, while loops, and do...while loops.

    These loops can help you iterate over arrays, perform a set of instructions multiple times, or execute a block of code until a certain condition is met.

    for (let i = 0; i < 5; i++) { console.log(i); }
  • Arrays
    • Arrays

    Arrays are used to store multiple values in a single variable.

    They are a special type of object that represents an ordered collection of elements.

    In JavaScript, arrays can hold different types of data, including numbers, strings, and even other arrays.

    You can access and modify array elements using their index.

    let fruits = ['apple', 'banana', 'cherry']; fruits.push('date');
